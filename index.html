I am currently working at Inria Paris, within the MiMove team.

My current research focuses on robust, large-scale machine learning under resource volatility. This work is part of the Cupseli project, a joint research initiative between Inria and Hivenet.

I am supervised by Nikolaos Georgantas (MiMove team, Inria Paris) and Davide Frey (WIDE team, Inria Rennes).

My research interests span machine learning, large-scale P2P networks, and robust, high-performance distributed systems.

Je travaille actuellement à Inria Paris au sein de l’équipe MiMove.

Mes travaux de recherche actuels portent sur l’apprentissage automatique robuste et à grande échelle dans un contexte de volatilité des ressources. Ces travaux s’inscrivent dans le cadre du projet Cupseli, une initiative de recherche conjointe entre Inria et Hivenet.

Je suis encadré par Nikolaos Georgantas (équipe MiMove, Inria Paris) et Davide Frey (équipe WIDE, Inria Rennes).

Mes centres d’intérêt de recherche couvrent l’apprentissage automatique, les réseaux pair-à-pair (P2P) à grande échelle, ainsi que les systèmes distribués robustes et haute performance.

Public GitHub Repositories:

  • Federated Learning for Occupancy Prediction with Highly Imbalanced, Non-IID, Low-Volume Data
    • Developed a working prototype to predict occupancy in BTIC (Atos Business Technology & Innovation Centre) building zones (distinct areas of the building) from device connection logs (network connection records for devices such as laptops and phones, aggregated every 15 minutes), using federated learning (models trained locally on each site; only model updates are shared, no raw data is exchanged).
  • Federated Learning with Differential Privacy for MNIST
    • Implemented federated learning with differential privacy for MNIST, allowing the model to be trained on decentralized data while preserving user privacy.
  • Neural Translation Model
    • Implemented a neural translation model using neural network architectures to translate from English to German.
  • Probabilistic Generative Models
    • Implemented probabilistic generative models to analyse and generate new data by learning probability distributions from a dataset.
  • Image Classifier for SVHN Dataset
    • Implemented an image classifier for the SVHN dataset, which accurately classifies images of house numbers captured in real-world environments.
  • Merkle Tree-Based File Integrity Verification System
    • Designed a system for verifying file integrity using Merkle trees with a Flask-based server and client application.
